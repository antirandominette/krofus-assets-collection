# krofus-assets-collection
A collection of Krosmaga and Dofus 2.71 assets.

### Tools used

All the files were retrieved with the help of these tools : 

- *[PyDofus](https://github.com/balciseri/PyDofus "PyDofus")* for Dofus
- *[Asset bundle Extractor](https://github.com/SeriousCache/UABE "Asset bundle Extractor")* for Krosmaga
